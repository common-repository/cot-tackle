Registers custom post types and custom fields for the Cot theme

== Description ==

**The Cot Tackle Plugin mainly develope to use in Cot theme**

This plugin registers custom post types and custom fields that are needed in the Cot Theme

== Installation ==

1. Go to Plugins > Add New and search for Cot Tackle. Install and activate it.
2. That's it. No configuration needed.
